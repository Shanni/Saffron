# Saffron Business Materials

This folder contains business and marketing materials for Saffron, an AI-powered crypto trading platform.

## Contents

### 1. Pitch Materials
- **saffron-pitch.md**: One-pager with key metrics and value proposition
- **pitch-script.md**: Detailed speaking script for presentations (5 minutes)
- **pitch-slides.md**: Complete slide deck content with visual descriptions

### 2. Landing Page
- **landing-page.html**: Complete HTML/CSS landing page for web deployment
- **landing-page-content.md**: Content structure and implementation notes

## Usage Instructions

### Viewing the Landing Page
1. Open `landing-page.html` in any modern web browser
2. For production deployment, upload to your web hosting service
3. Connect the email form to your preferred email marketing service (Mailchimp, etc.)

### Using Pitch Materials
- **For investors**: Use the complete pitch script and slides
- **For partners**: Focus on the one-pager and performance metrics
- **For team**: Use as reference for consistent messaging
